\section{コード（minted）}
minted はシンタックスハイライトに対応します（TeX Live 同梱の minted v3 /
latexminted により追加設定なしで動作）。インラインは \mintinline{python}{lambda x: x+1}。
\begin{minted}[linenos]{python}
def fib(n):
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a
\end{minted}
